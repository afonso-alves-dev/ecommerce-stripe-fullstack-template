
# 🛒 E-commerce Fullstack com Stripe

Template completo de **E-commerce Fullstack** com:

- React + Vite
- Node.js + Express
- Stripe Checkout
- Dashboard Admin
- API REST

 **portfólio Fullstack**.

---

# 🚀 Tecnologias

Frontend
- React
- Vite
- Axios

Backend
- Node.js
- Express

Pagamentos
- Stripe

---

# 📦 Instalação

## 1 Clonar projeto

git clone <repo>
cd ecommerce-stripe-template

---

# Backend

cd backend
npm install

Crie `.env`

PORT=5000
STRIPE_SECRET_KEY=sua_chave_stripe

Rodar servidor

npm run dev

---

# Frontend

cd frontend
npm install
npm run dev

---

# 💳 Fluxo Stripe

Carrinho → Checkout → Stripe → Sucesso
